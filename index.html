POS (Poin Of Sales)

Sebuah aplikasi Point of Sale (POS) berbasis web yang dikembangkan menggunakan Laravel sebagai framework backend dan Filament sebagai admin panel. Proyek ini dibuat dan dijalankan menggunakan Laragon sebagai web server lokal, serta Visual Studio Code sebagai code editor. Aplikasi ini membantu proses transaksi penjualan dengan pengelolaan data produk, perhitungan total pembayaran secara otomatis, dan manajemen transaksi agar lebih cepat, akurat, dan efisien.

  • Technologies: Laravel, Filament, PHP, MySQL, HTML, CSS, JavaScript, Laragon, Visual Studio Code
